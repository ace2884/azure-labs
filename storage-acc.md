***CREATING STORAGE ACCOUNT***

STEP 1 : Login to the Azure Portal
STEP 2 : Search Storage Account -> In the Left panel Object Storage -> Blob Storage -> Click 'Create'
STEP 3 : Give a Unique Name And Choose a region -> Click 'Review and Create'
STEP 3.1 : If it is ADLS Storage Account, After choosing name and region Click 'Next' and Enable the 
                   Hierarchical namespace -> Click 'Review and Create'

STEP 4 : After Deployment Click 'Go to resource' -> In the Left Panel Data Storage -> Container -> 
                +Create

STEP 5 : Click the created Container -> Click 'Upload' to upload a file OR Click 'Add Directory' to 
                create a folder



***Access Key Storage Account Level***

Left panel -> security And Networking -> access key -> key1 and key2 -> copy

***access key container level***

Left panel -> Data storage -> Containers -> three dots -> generate sas token -> copy
